# Variables de Template

## Variables générales
- `{{PROJECT_NAME}}` : Nom du projet
- `{{PROJECT_DESCRIPTION}}` : Description du projet
- `{{VERSION}}` : Version du projet
- `{{STATUS}}` : Statut (développement, production, etc.)
- `{{LAST_UPDATE}}` : Date de dernière mise à jour
- `{{LICENSE}}` : Type de license
- `{{LICENSE_DETAILS}}` : Détails de la license
- `{{MAINTAINER}}` : Mainteneur principal
- `{{EMAIL}}` : Email de contact
- `{{ISSUES_URL}}` : URL des issues GitHub
- `{{GENERATION_DATE}}` : Date de génération de la doc
- `{{LAST_ANALYSIS_DATE}}` : Date de dernière analyse

## Installation & Configuration
- `{{PREREQUISITES}}` : Prérequis système/logiciels
- `{{INSTALL_COMMANDS}}` : Commandes d'installation
- `{{CONFIG_COMMANDS}}` : Commandes de configuration
- `{{START_COMMANDS}}` : Commandes de démarrage
- `{{DEV_INSTALL_COMMANDS}}` : Commandes d'installation dev
- `{{TEST_COMMANDS}}` : Commandes de test
- `{{FORMAT_COMMANDS}}` : Commandes de formatage
- `{{DEV_START_COMMAND}}` : Commande de démarrage dev
- `{{PROD_START_COMMAND}}` : Commande de démarrage prod
- `{{DOCKER_COMMANDS}}` : Commandes Docker

## Structure
- `{{PROJECT_TREE}}` : Arborescence du projet
- `{{ARCHITECTURE_OVERVIEW}}` : Vue d'ensemble architecture
- `{{MAIN_COMPONENTS}}` : Liste des composants principaux
- `{{PATTERNS_USED}}` : Patterns identifiés

## API
- `{{API_BASE_URL}}` : URL de base de l'API
- `{{AUTHENTICATION_DETAILS}}` : Détails d'authentification
- `{{RESPONSE_FORMAT}}` : Format des réponses
- `{{ERROR_CODES}}` : Codes d'erreur
- `{{RESOURCE}}` : Nom de la ressource
- `{{RESOURCE_DESCRIPTION}}` : Description de la ressource
- `{{RESOURCE_PARAMS}}` : Paramètres de la ressource
- `{{RESOURCE_RESPONSE_EXAMPLE}}` : Exemple de réponse
- `{{TOKEN}}` : Token d'authentification

## Déploiement
- `{{DEPLOYMENT_PREREQUISITES}}` : Prérequis déploiement
- `{{DEPLOYMENT_CONFIG}}` : Configuration déploiement
- `{{BUILD_COMMANDS}}` : Commandes de build
- `{{DEPLOYMENT_TESTS}}` : Tests de déploiement
- `{{DEPLOYMENT_COMMANDS}}` : Commandes de déploiement
- `{{VERIFICATION_COMMANDS}}` : Commandes de vérification
- `{{CI_CD_PIPELINE}}` : Pipeline CI/CD
- `{{DEPLOYMENT_TRIGGERS}}` : Triggers de déploiement
- `{{ROLLBACK_PROCEDURE}}` : Procédure de rollback
- `{{HEALTH_CHECKS}}` : Health checks
- `{{DEPLOYMENT_METRICS}}` : Métriques de déploiement

## Environnements
- `{{DEV_ENVIRONMENT}}` : Environnement dev
- `{{STAGING_ENVIRONMENT}}` : Environnement staging
- `{{PROD_ENVIRONMENT}}` : Environnement production
- `{{ENV_FILE_EXAMPLE}}` : Exemple de fichier .env
- `{{ENVIRONMENT_SPECIFIC_CONFIG}}` : Configuration par env

## Mermaid Diagrams
- `{{ARCHITECTURE_DIAGRAM}}` : Diagramme d'architecture Mermaid

## Json Examples
- `{{MODEL_SCHEMA}}` : Schéma JSON d'un modèle
- `{{RESOURCE_CREATE_BODY}}` : Corps JSON pour création
- `{{RESOURCE_CREATE_DATA}}` : Données JSON pour création
